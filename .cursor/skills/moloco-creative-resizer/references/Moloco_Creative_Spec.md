# Moloco 广告素材设计与尺寸规范 (Moloco Creative Spec)

Moloco 作为一个全球知名的程序化广告平台，对投放的素材有着非常具体和严格的尺寸与格式规范。为了保证广告能获得最大曝光，请仔细参考以下通用模板规范：

## 一、图片素材（Image Creative）
常见于展示网盟与静态插屏。
* **通用/推荐尺寸**：
  * `300 x 250`（中等矩形，极其常用）
  * `320 x 50`（标准底部横幅）
  * `320 x 100`（大横幅）
  * `320 x 480`（全屏竖直 - Android 强力推荐）
  * `480 x 320`（全屏横向 - Android 强力推荐）
  * `728 x 90`（通栏横幅 - iOS 强力推荐）

## 二、原生广告素材（Native Creative）
必须包含应用图标、主图、标题和 CTA。
* **应用图标（App Icon）**：
  * `256 x 256` 像素以上（必须是严格的 `1:1` 方形，最大 500KB，推荐 PNG）
* **原生横图（Native Horizontal）**：
  * `1200 x 628`（业界最通用绝杀标准）
  * `1200 x 600`（**注意：投放韩国 Kakao 流量池必选该尺寸**）
  * `1200 x 222`
* **原生竖图（Native Vertical）**：
  * `720 x 1280`（核心竖版）
  * `720 x 960`
  * `1200 x 1600`
* **原生方图（Native Square）**：
  * `720 x 720`

## 三、视频素材规格（Video Creative）
通常用于应用内的全屏视频展示（如激励视频），并附带片尾落版图(End Card)。
* **视频画幅**：必须严格为 `16:9`（横屏）或 `9:16`（竖屏），**决不能在边缘留黑边凑尺寸**。
* **时长建议**：必须准备不同时长的版本组合策略（如 16~30 秒，31~60 秒起步）。
* **格式与大小**：MP4格式，不得超过 10 MB。

## 💡 最佳提审实践
如果你是一名游戏/应用买量推手，最万金油的**首发高优先级图片套装**组合是：
1. `1200x628`（横图主体）
2. `1200x600`（用于 Kakao 渠道补偿）
3. `720x1280` / `1200x1600`（高质量竖版）
4. `720x720`（全能正方形适配）
5. `300x250`（信息流中距图）
