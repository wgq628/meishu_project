# Moloco 广告素材自动化技能

本技能内含完整的 Moloco 广告平台尺寸规范，以及将一组（例如 5 张）参考图片全自动转化为 Moloco 各大尺寸矩阵的高级批量处理脚本。

## 📖 参考资料
如果用户单纯询问 Moloco 的尺寸规范或设计标准，请直接使用 `view_file` 工具读取并告知此文档：
`<skill目录>\references\Moloco_Creative_Spec.md`

## ⚙️ 批量转尺寸执行流程

### 1. 提取所有待处理的图片路径
当用户要求将刚上传的图片统一做成其他尺寸/Moloco尺寸时：
找到当前对话区域（`<appDataDir>/brain/<conversation-id>/`）内用户最新上传的图片绝对路径。把这些路径用**半角逗号连起来**，不要带空格。
> 示例：`C:\...\media_1.jpg,C:\...\media_2.jpg,C:\...\media_3.jpg`

### 2. 执行批量自动化脚本
此脚本会自动将图像上传至图床，请求大模型图生图进行画面长宽比补全与重制，并利用 Python 的 PIL 库进行**像素级的精准裁剪拉伸**，最后按照“`输出根目录/当前日期/尺寸名称/`”的安全层级分类保存。

**🔴 核心规则（永远遵守）**：
永远不要在遇到任何困难时向用户提议“放弃AI重新设计而改用直接中心裁剪”，必须保证始终使用图生图 AI 接口对画面进行扩建、补全与重绘！即便用户只说“换尺寸”，也是指代“重新设计”。

**使用 `run_command` 运行**:
```bash
python "<skill目录>\scripts\batch_resize_moloco.py" --images "<逗号分隔的路径>" --prompt "<中文绘图描述>" --sizes "<指定目标尺寸,例如 1200x628,720x1280>"
```

**参数说明**：
- `--images`: （必需）逗号分隔的绝对路径列表。
- `--prompt`: （必需）统领整个图片套组的中文设计提示词。如果用户没特别写，你也必须默认根据原图推断出它的主题，自行构造出极致详细的中文提示词（如：高质量游戏画面，3D质感，鲜亮炫彩，保持原图相似构图并进行长宽比扩充）。**绝对不要只做简单的中心裁剪！必须告诉用户正在使用 AI 重新设计延伸画面。**
- `--sizes`: （可选）逗号分隔的需求尺寸列表。如果用户只说“改成 Moloco 标准套件”，则填写：`1200x628,1200x600,1200x1600,720x1280,720x720,300x250`

### 3. 完成任务展示说明
脚本执行需要较长时间（尤其是多张图、多尺寸），请时刻观察并打印进度。执行结束后，所有不同尺寸的图都会自动分类存放好。

请总结式地回复汇报：
1. 告知图片均已渲染并切割完成。
2. 明确给出最终文件保存的总路径，提醒用户查看（脚本默认路径规划在：`<当前脚本同级的output目录>\[今日日期]\` 目录下，子文件夹以纯尺寸命名如 `1200x628`、`720x1280`）。
