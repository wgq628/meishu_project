# 设计商店宣传图 (Store Screenshots / 五图)

**技能定位**：
专注于策划和生成适用于各大应用商店（App Store / Google Play）的垂直竖版推广宣传五图，以及适用于抖音等渠道的买量核心视觉。

## 1. 尺寸硬性要求
所有的商店图及背景图资产必须默认输出或指定为 **9:16 竖版高清图**。

## 2. 情绪引导策略 (五图标准公式)
每张图必须带有极强的单点突破情绪，不可平庸：
- **图1 (核心玩法/强迫症拯救)**：标语如“分类所有物品！”。展示一个凌乱的 3D 货柜和发光的消除组合光束，带明显的拖拽指示。
- **图2 (极端混乱/压力挑战)**：标语如“你能挑战第100关吗？”。展示超大型或被遮挡得密密麻麻的超级货柜，色彩缤纷，但错乱感极高，触发挑战欲。
- **图3 (解压消除/视觉爽感)**：标语如“连击特效爽翻天！”。满屏的绚丽消除粒子特效，商品集体缩小消失的光斑动画截图。
- **图4 (装扮/扩展玩法)**：标语如“收集盲盒布置你的超市”。展示玩家除消除外，收集特殊玩具或装扮专属超市大厅的界面。
- **图5 (图鉴库/海量物品)**：展现琳琅满目的超精美 3D 商品海报合集，强调游戏的收藏乐趣。

## 3. 产出模式
可根据要求生成全套文案策划表，也可单张提供画图拆解提示词或宣发大字占位模板。
