## 🚀 如何使用 (How to Use)
与Agents对话
1. **帮我制作货柜排序的木制货柜，竖版3X7结构，给我1-20套参考**
2. **上传参考图，根据参考元素/风格/排版/制作烧烤三消的食物素材，要5组不一样的食物**
3. **上传1-5张参考图，根据参考图完成1-5套moloco素材所有尺寸图像**
4. **上传需求描述截图，根据需求描述制作你的版本**


**查阅配置**：进入对应技能文件夹，阅读 `SKILL.md` 了解触发条件与调用逻辑。
**执行脚本**：部分技能附带 `scripts/` 目录，可通过系统环境运行相关 Python/Shell 脚本。
**调用 MCP**：如 `Banana2-MCP` 类技能，可通过注册 MCP 服务直接在对话中触发。

## 🎨 设计准则 (Design Philosophy)

我们追求 **"高效、精致、专业"** 的 AI 产出。所有设计类技能均遵循现代审美趋势，包含但不限于：
- 视觉输出：融入现代光影、渐变、细腻质感，拒绝粗糙低质效果
- 行业合规：严格贴合应用商店、广告平台、游戏设计行业尺寸与规范
- 使用体验：简化交互逻辑，指令直白易懂，兼顾新手上手与专业效率
- 标准化：统一模块结构，方便后续扩展、维护与团队协作复用
